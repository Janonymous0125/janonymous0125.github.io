Ethical Hacking Notes

'Hacking' in general is to find bug in a system and exploit them to get what we want OR to manipulate the system to do what we want which it is not built to do.

Different types of Hackers

Mainly Hackers are grouped together into three different groups. The most obvious difference between them are their Ethical guidelines.

Bandit:OverTheWire

Bandit is a CTF base Wargame, Like other CTF you will need to catch the flag and use it as the password for the next level. I highly encurage beginners to start their penertration testing journey here. Click Here to enter to the Bandit's page for more details.

Bandit Introduction

Level 0 : Initial Login

Level 0 is the start of this game and the way you will be able to interact with host machine. From the goals that the creator of Bandit, we know that we have to log into the game via OpenSSH.

Bandit0 Goals

From the above, we can gather the infomation below:

  • Host: bandit.labs.overthewire.org
  • Running port: 2220
  • User: bandit0
  • Password: bandit0

The credentials of user bandit0 is already given, the host and port of the game is given as well. All we need is to log into the game using the infomation given to start. The format of the ssh command is at the following: 

ssh -p {port number} {user}@{host}
Combined with the given infomation, we get this: 
ssh -p 2220 bandit0@bandit.labs.overthewire.org

After entering the command, enter the password; bandit0 which is given by the creator in the goal of the level. You have successfully beated level 0! Please read the rules and do not break them...

Level 0 Completed

Level 0 → Level 1 : Reading Files with CLI through OpenSSH

Bandit1 Goals

From the above, we can gather the infomation below:

  • The password is stored in the readme file.
  • The readme file is located in the home directory of the current level. (bandit0)
  • We need to use the password from the readme file to login to the next level.
  • Hint: "bandit1" is the username for the next level.

Continuing from last level, we can find our directory using pwd command. pwd stands for Print Working Directory. This helps us to know where we are and where we are workinng on. 

bandit0@bandit:~$ pwd
/home/bandit0

From the output of the pwd command, we know that we are already in the home directory of the current level. Then by running the ls command, this will list all the files in the current directory. 

bandit0@bandit:~$ ls
readme

Now that we know that the readme file contains the password for the next level. There are two main ways to retrieve the password from the file. Use of CLI Tools like 'cat' OR text editor. For example:

bandit0@bandit:~$ vi readme

'vim' is a IMproved version of vi. Similar to vi but with even more tools added to the commands

bandit0@bandit:~$ vim readme

Bandit1 Complete

After we get the password please note it down somewhere you can reach easily... Taking notes will let you know where you left off and you can log back in.... if you forgotten the password or you didnt note it somewhere in your local machine you might need to start over again.
