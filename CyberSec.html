Linux Skills

Linux has a huge library of distributions. I've been using BlackArch Linux, Kali Linux, Parrot Sec OS and Ubuntu Linux during my learning curve. Among all the distro that I've used, Kali Linux and ParrotSec OS is my go to, mainly it is already builtin with all the tools needed to start your ethical hacking journey. As for Ubuntu and BlackArch Linux, its more customizable to do other stuff like running an AI Language Model, Web Application, and more advanced tools that you can code yourself.
